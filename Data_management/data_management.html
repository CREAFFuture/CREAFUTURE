Resources to organize your scripts, improve your workflow, etc

  • A beginner’s guide to R, R Studio, and the tidyverse.
  • Primers on data visualization, iteration, writing functions, reproducibility, etc.
  • Recommendations on style, appropaite naming of objects, etc. You can try the styler package to implement a consistent style.
  • Don’t repeat yourself (DRY). Write functions instead of copy-pasting code.
  • Rodríguez-Sánchez, F. 2020. Quince consejos para mejorar nuestro código y flujo de trabajo con R. Ecosistemas 29(3): 2129.
  • You can also read this Twitter thread.
  • A wealth of resources.
  • Keep track of all the parentheses in your code by colouring each pair of brackets in a different color: Tools > Global options > Code > Display > Use rainbow parentheses
  • A comparison across repositories (Dryad vs Zenodo vs Figshare vs others).
  • Some recommendations on how to name your files, specially relevant for manuscripts.
  • Tabular data may be in long or wide format, the switch between the two is called “pivotting”, of which you can change a nice animation here.
